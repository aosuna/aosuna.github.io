Virtual Machine and Assembler


Virtual Machine and Assembler is a University Project that was written in C++. It uses various programming techniques such as friend classes, pointers and various data structures. This project is composed of 3 parts.

  • The first is the Assembler, the assember is a set of instructions that can be executed, and it outputs object code to a file that is read in by the Virtual Machine.
  • Second part of the project was the have the Virtual Machine read in the object code and execute the instrucitons.
  • The thrid part of the project was to build a Process Control Block or PCB that maintined all the processes environment variables, cpu time, etc to allow for context swithing between different running process.

The operating system brought all these componets together and ran the results in ready and wait queues, kept track of the time to execute and handled reading and writing to and from a file of all processes.
